Meaning, tone, and nuance can be lost or distorted — particularly for idiomatic expressions, specialised terminology (legal, medical, technical), and low-resource languages. Do not rely on automated translations for critical decisions, official documents, or medical/legal advice. When accuracy matters, always verify with a qualified human translator.
🔒 Privacy policy

Vernac collects no personal data. It does not track your browsing, log translated text, or send information to any external server beyond the translation API itself — which goes directly from your browser to Google or DeepL, not via any Vernac-operated server.

All preferences are stored locally in your browser and never leave your device. Your DeepL API key is stored only for the current browser session and is only ever transmitted directly to DeepL's API.

No analytics. No telemetry. No third-party trackers.

How it works

Vernac offers three ways to translate. Requests go directly from your browser to the API — no intermediate server.

Paste Mode
Type or paste text directly into the input box and translate instantly.
Selection Mode
Highlight text on any page — a Translate button appears. Click it or right-click and choose Translate with Vernac. The result opens in a sidebar so you can read original and translation side by side.
Scan Page
Extracts all readable content and translates section by section. Click any chunk to jump to its source on the page, and vice versa.
Permissions explained

Vernac requests <all_urls> access so it can inject the floating translate button and page-scanning into any site you visit. The extension only reads page content when you explicitly trigger a scan or selection — it does not run continuously in the background.

activeTab — reads the current tab for page scanning.
storage — saves your preferences locally.
contextMenus — adds the right-click translate option.
scripting — injects the content script for selection mode.
tabs — closes open sidebars across all tabs when switching to popup mode.
alarms — keeps the background script alive in Firefox to maintain sidebar state.
